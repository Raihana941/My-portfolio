Grand Cinema

A sleek and immersive cinema-themed website that captures the excitement of the big screen. This project mimics the layout of a real movie theatre interface, complete with a dramatic design and subtle animations. Built entirely with HTML and CSS, it demonstrates my ability to create engaging, visually rich experiences using only front-end code.

Sunny Mall

A bright, modern landing page for a fictional shopping mall, designed with a clean layout and playful color palette. Sunny Mall features a stylish interface with intuitive navigation, making the digital mall experience feel light and welcoming. This project highlights my creativity in layout design and my eye for aesthetic harmony using pure CSS styling.

Arg Restaurant

A refined restaurant website concept inspired by elegance and authenticity. The layout includes beautifully structured menu sections, a welcoming hero banner, and a classic restaurant feel. Built with semantic HTML and styled entirely with CSS, this project shows my ability to design user-friendly pages that reflect a brand’s personality.

Persian Proverbs

A poetic, minimalistic website that presents timeless Persian wisdom through beautiful typography and design. Each proverb is highlighted with elegant spacing, floral touches, and soft colors that honor the cultural essence of the content. This project showcases my focus on clean aesthetics, typographic balance, and culturally inspired design.
